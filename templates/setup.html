What is a Presenter ID?

To file anything to Companies House via software (rather than their website), you need a Presenter Account. This gives you two credentials:

  • Presenter ID — your identity as a software filer (11 characters)
  • Presenter Authentication Code — your password for the gateway (11 characters, letters and numbers)

These are separate from the company authentication code. The company auth code proves you're authorised to file for that specific company. The presenter credentials prove you're an authorised software filer.

Step 1: Get a Test Account

Before you can file for real, Companies House wants you to test your software. Email them to request a test account:

Email: xml@companieshouse.gov.uk

Subject: Request for XML Gateway test account

Include:

  • Your name / organisation name
  • Contact name
  • Address
  • Email address
  • Telephone number

They will reply with a test Presenter ID and test Authentication Code. Turnaround is typically a few working days.

Before emailing, you should have read the technical interface specifications (this app already implements them).

Step 2: Test Your Filing

Once you have test credentials:

  1. Start the app with your test credentials: 
    export CH_PRESENTER_ID="your_test_id"
export CH_PRESENTER_AUTH="your_test_auth"
./start.sh
  2. Fill in the form (it's pre-filled with the right numbers)
  3. Click Validate & Preview
  4. Check the iXBRL document and GovTalk envelope look correct
  5. Click Submit Filing — this goes to CH in test mode (GatewayTest=1)
  6. Email xml@companieshouse.gov.uk to tell them your test submission is ready for manual review
  7. Wait for their confirmation that your test submission looks good

Step 3: Get a Live Presenter Account

Once testing is approved, you need live credentials. Since we're only filing accounts (no fee-bearing documents), you can use the quick online route:

Option A — Online (recommended, near-instant):

Apply at: https://find-and-update.company-information.service.gov.uk/presenter-account

  • Sign in or create a Companies House account
  • Provide your name, correspondence address, business name or company number
  • Tick "software filing" when asked
  • Credentials issued almost immediately

This is for non-fee-bearing documents only (which accounts are). Perfect for our use case.

Option B — Credit account (if ever needed for fee-bearing filings):

Download the application form from: GOV.UK credit account page

  • Fill in the PDF form (tick "software filing" in section 8)
  • Email to chdfinance@companieshouse.gov.uk
  • Takes up to 5 working days
  • Credentials sent by post in separate letters for security

Step 4: File for Real

Once you have live credentials:

export CH_PRESENTER_ID="your_live_id"
export CH_PRESENTER_AUTH="your_live_auth"
export CH_ENVIRONMENT=live
./start.sh

Then go through the normal form → preview → submit flow. The filing will be sent to Companies House for real.

Credential Format Reference

Presenter ID11 characters. Starts/ends with 000 (credit account) or starts with E followed by 10 digits (presenter-only account)
Presenter Auth Code11 characters, uppercase letters and numbers. Example: AC75D45JUQA
Company Auth Code6 characters, alphanumeric. Posted to your registered office annually by Companies House

Note: Future of the XML Gateway

Companies House has said they plan to replace the XML Gateway with a REST/JSON API in the long term. No definite timeline has been given. For now, the XML Gateway is the only way to software-file accounts, and this app uses it.
